#Movies Recommender System
This project is a Content-Based Movie Recommender System developed using Python, Machine Learning, and Natural Language Processing (NLP) techniques. The system recommends movies similar to a user-selected movie by analyzing important movie attributes such as genres, keywords, cast, crew, and overview from the TMDB 5000 Movies Dataset. Data preprocessing and feature engineering techniques were applied to clean and transform the dataset, including extracting top cast members, directors, genres, and keywords. The textual data was combined into tags and converted into numerical vectors using CountVectorizer, while Porter Stemmer was used for stemming and improving text consistency. Cosine Similarity was then used to calculate similarity scores between movies and generate accurate recommendations. The project demonstrates practical implementation of content filtering, vectorization, NLP preprocessing, and recommendation algorithms using libraries such as Pandas, NumPy, Scikit-learn, NLTK, Matplotlib, and Pickle.
